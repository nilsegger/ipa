Beobachter Form
Der Name darf nicht leer sein und muss mindestens 3 bis 100 Charakter haben.
Die Art darf nicht leer sein.
  1. Ein Zählerstand Beobachter misst wie oft ein Sensor einen Wert geschickt hat und löst beim Auslöserwert eine Meldung aus.
  2. Ein Richtwert darüber gibt eine Meldung aus, wenn der mit Wert Name definierte Wert eines Sensor über dem Auslöserwert liegt.
  3. Ein Richtwert darunter gibt eine Meldung aus, wenn der mit Wert Name definierte Wert eines Sensor unter dem Auslöserwert liegt.
Der Sensor darf nicht leer sein.
  1. Der Adeunis Sensor meldet den Wert 'temperature'.
  2. Ein Elsys ERS CO2 Sensor besitzt die Werte 'temperature', 'co2', 'motion', 'light' und 'humidity'.
  3. Der Tabs Sensor kann nur mit dem Zählerstand verbunden werden.
Der Auslöserwert darf nicht leer sein.
Der Wert Name darf nur bei einem Zählerstand leer sein.
Dieses Feld muss nur für Richtwert Beobachter ausgefüllt werden.
Der Stand darf nicht leer sein.
Dieses Feld muss nur für den Zählerstand Beobachter ausgefüllt werden.
